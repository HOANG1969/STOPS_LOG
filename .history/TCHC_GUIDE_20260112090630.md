### 2. TCHC Manager (Lãnh đạo TCHC)
- **Email**: `tchc.manager@pvgas.com.vn`  
- **Password**: `123456`

- **Position**: Trưởng phòng TCHC

## Workflow TCHC hoàn chỉnh

1. **Employee** tạo yêu cầu → Status: `draft` → `pending`
2. **Trưởng bộ phận** phê duyệt → Status: `approved` ("Trưởng bộ phận phê duyệt")
3. **TCHC Checker** kiểm tra → Status: `tchc_checked` ("Nhân sự TCHC đã kiểm tra")
4. **TCHC Manager** phê duyệt cuối → Status: `tchc_approved` ("Lãnh đạo TCHC phê duyệt")

## Cách tạo thêm user TCHC

### Option 1: Qua Admin Panel
1. Login với tài khoản Admin
2. Vào "Quản lý người dùng" 
3. Tạo user mới với department = "TCHC"
4. Tick vào checkbox:
   - `is_tchc_checker` cho nhân sự TCHC
   - `is_tchc_manager` cho lãnh đạo TCHC

### Option 2: Qua Database
```sql
-- Tạo TCHC Checker
UPDATE users SET is_tchc_checker = 1 WHERE email = 'user@example.com';

-- Tạo TCHC Manager  
UPDATE users SET is_tchc_manager = 1 WHERE email = 'manager@example.com';
```

### Option 3: Chạy Seeder
```bash
php artisan db:seed --class=TchcUserSeeder
```
